SE01 Project

This is my project for the SE01 module. It's a small website with three pages built using HTML and CSS.

The pages are:
- index.html (home page)
- about.html (about page)
- contact.html (contact page with a form)

All pages are linked together through a navigation bar at the top.

I used media queries to make the layout responsive. There are two breakpoints at 600px and 900px so the site works on mobile, tablet and desktop.
